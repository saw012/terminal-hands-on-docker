# terminal-hands-on-docker

## dockerをインストールする
- [Get Docker](https://docs.docker.com/get-docker/)を参考に、dockerをインストールします。

## dockerイメージをローカルに落とす
github container registryからイメージを取得する

```bash
docker pull XXX
```

## dockerイメージを実行する
```bash
docker run -it XXX zsh
```

## ハンズオンの内容を実施する
